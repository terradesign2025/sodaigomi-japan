# フィードバックフォーム設定書
## Googleフォーム作成手順

### フォームタイトル
「粗大ごみ判定アプリ — 品目が見つからない？教えてください」

### フォームの説明文
```
このアプリをより便利にするために、皆様のご協力をお願いしています。
「検索しても出てこなかった」「この言葉でも出るようにしてほしい」など、
どんな小さな気づきでも大歓迎です！

所要時間：約1分
個人情報の入力は不要です
```

---

## 質問項目（この順番でコピーして作成してください）

### Q1: 種別（必須・ラジオボタン）
タイトル: 「お問い合わせの種類を選んでください」

選択肢:
- 🔍 品目が見つからなかった（検索してもヒットしなかった）
- 💡 こんな言葉でも検索できるようにしてほしい（同義語の提案）
- ⚠️ 情報が間違っている・古い（料金・ルールの修正）
- 🐛 アプリが正しく動かない（バグ報告）
- 💬 その他のご意見・ご要望

---

### Q2: 検索したキーワード（必須・短文入力）
タイトル: 「何を検索しましたか？入力した言葉をそのまま書いてください」
説明: 「例：ふとん、ソファ、デスク、エレキギター など」
プレースホルダー: 例）ふかふかいす、ゲーミングチェア

---

### Q3: 探していた品目（必須・短文入力）
タイトル: 「本当は何を捨てたかったですか？」
説明: 「正式名称でなくても大丈夫です」
プレースホルダー: 例）椅子、ソファ、机、ベッド

---

### Q4: 提案する言葉・同義語（任意・段落テキスト）
タイトル: 「この品目をどんな言葉で検索する人が多いと思いますか？」
説明: 「複数ある場合は改行して入力してください」
プレースホルダー:
例）
ゲーミングチェア
ゲーミング椅子
PCチェア
オフィスチェア

---

### Q5: お住まいの地域（任意・プルダウン）
タイトル: 「お住まいの市区町村（任意）」
説明: 「データ追加の優先順位に使用します」
選択肢: 八王子市 / その他（東京都） / 神奈川県 / 埼玉県 / 千葉県 / その他の関東 / 関東以外

---

### Q6: その他コメント（任意・段落テキスト）
タイトル: 「その他、気づいたこと・ご要望があればご自由にどうぞ」
プレースホルダー: アプリへのご要望、気になった点など何でも

---

## フォーム設定

### 確認メッセージ（フォーム送信後に表示）
```
ありがとうございます！🙏

いただいたご意見は、アプリの改善に役立てます。
同義語の追加・修正は随時反映していきます。

引き続き全国粗大ごみ判定アプリをよろしくお願いします。
— Terra Design
```

### その他設定
- [ ] メールアドレスの収集: しない
- [ ] 回答のコピーを送信: なし
- [x] 送信後に新しい回答を送信できるようにする: ON
- [ ] ログイン必須: なし（誰でも回答できる）

---

## 完成後にやること

1. フォームの「回答」タブ → スプレッドシートにリンク
   シート名: `フィードバック受信`
2. フォームURLをコピーして下記のGASの `FEEDBACK_FORM_URL` に貼り付け
3. apps/feedback-widget.html の `FORM_URL` に貼り付け
