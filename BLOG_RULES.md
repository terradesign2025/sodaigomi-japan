## 1. 記事HTMLテンプレート仕様

### head内（全記事必須）
- title: 28〜35字目安・主要KW（品目名/地域名＋「粗大ごみ」「処分」「料金」）を前方に・【2026年◯月版】等の年号入り・末尾「｜粗大ごみナビ」
- meta description: 120〜160字・数値（料金幅・品目数）を1つ以上入れる
- canonical: `https://sodaigomi-navi.com/blog/[slug].html`（**旧 terradesign2025.github.io は使用禁止**）
- OGP一式（og:title/description/type=article/url/site_name=粗大ごみナビ/image=ogp.png 1200×630）＋twitter:card
- favicon 3種（favicon.svg/favicon.ico/apple-touch-icon.png）
- GA4: G-4KRRV4LLLH

### JSON-LD（2〜4本）
- Article（必須）: headline/datePublished/dateModified/author=Organization「株式会社テラデザイン」/publisher/description
- FAQPage（必須）: 本文末尾のFAQと文言を完全一致させて5問程度
- BreadcrumbList（必須）: トップ›ブログ›記事名
- HowTo（手順系記事のみ）

### 本文構成
1. hero（カテゴリバッジ＋h1＋日付・著者「テラデザイン編集部」＋要約文）
2. パンくず → 目次（アンカーol・見出しは質問形）
3. h2×5〜6本（各h2直下に結論の一文 or definition-box）
4. 中間CTA（本文1/3〜1/2地点）→ 記事末CTA → FAQ×5 → 関連記事カード → footer
5. 装飾パーツは既存記事の definition-box/step-flow/table-wrap/warning-box/note-box/faq-item/cta-box/related-card を再利用

## 2. 文体ルール

- ですます調・語りかけ型・PREP法標準（結論ファースト）。PASONAは使わない（行政情報ジャンルのため煽り不要）
- 「！」多用・「最強」「圧倒的」等の煽り表現禁止。絵文字はUIアイコン用途のみ
- 1文短く、2〜3文ごとに段落。専門用語は初出時に補足

### 断定OK/NGの線引き（このジャンルの核心）
| 区分 | 扱い | 例 |
|---|---|---|
| 自治体の公式料金・受付方法・法律 | 断定OK（公式情報が出典） | 「八王子市のソファー（70cm未満）は500円です」 |
| 自治体差がある事柄 | 断定禁止・幅表現＋確認誘導 | 「30cm以上が目安ですが、基準は自治体により異なります」 |
| 所要日数などの経験則 | 推量表現 | 「一般的に1〜2週間程度かかることが多いです」 |
| お得さ・効果の主張 | 控えめに。「必ず安くなる」禁止 | 「半額程度になる自治体もあります」 |
| 自社アプリの機能・対応数 | 事実の数値は断定OK・**公開時点の実数に毎回更新**（現在478市区） | |

### 免責（料金に触れた記事は必ず）
> 掲載している料金・申込方法は各自治体の公式情報をもとに作成していますが、改定等により実際と異なる場合があります。お申し込み前に必ず各自治体の公式情報をご確認ください。

## 3. SEO/AIO公開前チェックリスト

- [ ] title/meta description/canonical/OGP/favicon/GA4（上記仕様どおり）
- [ ] h1は1つ・h2は質問形または結論先出し形（AEO対応）
- [ ] FAQ5問＋FAQPage JSON-LD（本文と文言一致）
- [ ] 内部リンク: ①アプリCTA×2（`/?city=コード` 推奨）②関連記事カード×1以上 ③市区ページ `/city/XXXXX.html` へ本文中2〜4本
- [ ] **ローカルテキスト**: 具体的な市区の料金実例を本文に1〜2件入れ、該当市区ページへリンク（「地域名×品目×料金」でAIに引用されやすくする）
- [ ] 料金には「※2026年◯月時点」を必ず併記
- [ ] 曖昧表現の数値化（「安い」→「200円〜2,900円」）
- [ ] noindexなし・canonical本番ドメイン確認

## 5. 禁止事項

1. **未実装機能への言及禁止** — LINE連携・写真AI判定等は実装まで書かない。書けるのは「市区と品目を選んで料金を検索・合計」のみ（誇大表現は景表法の優良誤認リスク）
2. **他社サービスの実名プロモ禁止** — 特定の回収業者・買取店・フリマの推奨リンクをしない。一般名詞（「不用品回収業者」「フリマアプリ」）に留める。スポンサー（回収業者等）との利益相反にも注意。公的機関（環境省・自治体・国民生活センター）へのリンクはOK
3. **料金の時点表記なし記載禁止**
4. **コピペ由来の未検証事実禁止** — NotebookLM等の出力は原典（自治体・公的機関）で裏取りしてから。料金は自社cities/データのみを正とする
5. **旧URL（terradesign2025.github.io）使用禁止**
6. **「参考元」「出典」「ソース元」「参照記事」等の記載禁止**（2026-07-10オーナー指示）— 裏取りは必ず行うが、情報源の列挙・出典セクションは記事にもObsidian保存物にも書かない。読者の手続きに必要な機能的リンク（料金検索ページ等）と内部の関連記事リンクは可

## 6. 執筆優先度（BLOOM評価・2026-07-09時点）

- **S**: 品目別大型家具／収集不可・家電リサイクル

- **B**: 都市別（スポンサー営業と連動して優先順を決める）／小型家電／申込み手順／フリマ使い分け
- **C・保留**: アプリ誘導系（LINE/AI実装後のローンチ弾として温存）／季節（公開2か月前に仕込み）
